Run `EasyQuantizationGUI.bat` to start the application.

Requirements:
- [Python](https://www.python.org/downloads/windows/)
- Windows (can be adjusted later to support Linux)

Supported Model Architectures
-----------------------------

This tool supports quantization for a wide range of model architectures commonly used in image generation and related tasks. The following architectures are supported:

- Flux
- SD3
- Aurora
- HiDream
- Cosmos Predict 2
- Hyvid
- Wan
- LTXV
- SDXL
- SD1
- Lumina 2

Usage
-----

Run `EasyQuantizationGUI.bat` to start the application.
